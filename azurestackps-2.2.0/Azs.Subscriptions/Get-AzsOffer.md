# Get-AzsOffer

## SYNOPSIS
Get the list of public offers for the root provider.

## SYNTAX

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Get the list of public offers for the root provider.
