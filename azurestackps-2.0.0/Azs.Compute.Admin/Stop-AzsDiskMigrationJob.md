# Stop-AzsDiskMigrationJob

## SYNOPSIS
Cancel a disk migration job.

## SYNTAX

```
Stop-AzsDiskMigrationJob -Name <String> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Cancel a disk migration job.

## EXAMPLES
